# Single source of truth for every external download used by warmup.bat,
# build.bat, and make-slim-ffmpeg.bat/.sh. Format is plain KEY=VALUE so
# that both cmd (via scripts\load-downloads.bat) and bash (via
# `set -a; . scripts/downloads.txt; set +a`) can read it without a parser.
#
# To bump a tool version: edit the URL_* and FILE_* lines together, then
# run scripts\check-downloads.bat to confirm the new pointer is reachable
# before the next warmup. No URL belongs in any other file.
